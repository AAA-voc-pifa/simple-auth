# 开发文档

Simple Auth 是一个简单的用户、会话系统。（以下简称 SA）

用户可通过邮箱、OAuth 验证身份。支持多种 OAuth provider，如 GitHub、Google。

不区分登录和注册，“第一次登录”视为注册。

用户只能通过 OAuth 注册（邮箱的“真实度”较低）。
注册之后，用户可以绑定邮箱，之后可以通过邮箱验证码登录。（没有“邮箱+密码”登录）

用户可能不记得第一次登录用的是 GitHub，还是 Google。
因此，每当用户第一次通过 OAuth 验证后（即 SA 拿到 OAuth 提供的 userid 之后），
SA 不立刻注册用户，而是告诉客户端“这可能是一个新用户，你问问用户，是使用其他账户登录，还是注册新用户”，并且把当前 OAuth 信息以 jwt 的形式响应给客户端（有效期较短）。
